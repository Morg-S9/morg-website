After that, I decided to leave it alone for a bit. However just yesterday, I decided to look into hosting providers, and found Firebase Hosting.

Firebase Hosting is a service of Firebase, a service created by Google for back-end'ing web and mobile apps.
It does have a free plan, albeit with a quota of 360mb per day, but my website is fairly small, and Cloudflare Caching probably helps a bit, so it would work for me.
Setting it up was rather easy actually. Just download the CLI, login, initalize the directory you keep the project in, and deploy it.
You can also link custom domain names for free, which is pretty nice.

So if you're like me, just making a small website to express yourself on, (and if you don't mind getting your hands a little dirty cough cough command prompt cough cough coding your own website cough cough), I'd 100% recommend Firebase. Check them out here.
